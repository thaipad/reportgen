Сопроводительная записка к заданию "написать генератор текстовых отчётов".


Сделал согласно требованиям.

Дополнения:
1. В случае рассогласования общей ширины страницы и суммы колонок,
выдаю предупреждение в консоль и игнорирую заданную ширину страницы.
2. Не использую строку-разделитель "-----" перед шапкой и после последней записи на странице.
3. Генератор реализовал в виде jar-файла (testreport-1.0.jar).
Пример вызова: java -jar testreport-1.0.jar settings.xml source-data.tsv example-report.txt
Дополнительно для ОС Windows приложил файл run.cmd (run.cmd settings.xml source-data.tsv example-report.txt).
4. Допускается при запуске опустить 3-й параметр. 
Тогда отчет запишется в файл report_YYYYmmddHHMM.txt в домашнюю папку. 
5. Версия java должна быть не ниже 8.
